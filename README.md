CC租车系统
====
新手上路，按照暑期实践项目要求设计的第一个系统。<br>
主要功能：<br>
客户——注册、登陆、查看个人信息、修改个人信息、查看订单、租车、还车<br>
员工——添加优惠券信息、报废车辆<br>
管理员——员工信息管理、网点信息管理、用户订单查看、车辆租用信息查看、车型信息增删改、车辆信息增删改、车类信息增删改<br>
<br><br>
`编译软件：IntelliJ IDEA 2021.2`<br>
`使用插件：JFormDesigner`<br>
